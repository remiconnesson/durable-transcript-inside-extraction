# Rendering Strategies for MDX Blogs

When building a developer blog, choose a rendering strategy based on how often content changes.

## Practical rule of thumb

1. Use **SSG** for stable in-repo content.
2. Add **ISR** when updates should appear without full redeploy.
3. Use **SSR** for authenticated or personalized variants.

For this template, we start with static generation and finite slugs.
